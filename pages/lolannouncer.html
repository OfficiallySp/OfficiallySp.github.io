LoL Announcer

Custom Announcers for League of Legends Made Easy

The LoL Announcer project was a simple and easy way to use custom announcers in League Of Legends. It used the Live Client Data API to get the game state and then played a custom announcer based on the game state. Unfortunately, due to Riot Vanguard and the changes it has made to the game, this project is no longer functional.
