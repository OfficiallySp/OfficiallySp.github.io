Destiny Tools

6 in 1 Destiny 2 Toolset

Destiny Tools is a collection of tools for Destiny 2. it supports the following,

  • 1. Destiny Item Manager (DIM)
  • 2. light.gg
  • 3. trials.report
  • 4. Destiny Recipies
  • 5. D2 Armor Picker
  • 6. D2 Checklist

All of these are switchable via hotkeys or by using the menu bar
